\doxysubsection{Detailed Description}
The wav class. 

This class contains all the functions related to the wav file (excluding anything metadata related, I didn\textquotesingle{}t have time). It inherits from the \mbox{\hyperlink{class_i_save_file}{ISave\+File}} class and overrides the write\+File and check\+If\+File\+Exists functions.
